HTML语法基础

HTML语法基础

学习django的时候发现还是要懂点html语法的

语法

  • html是一种展示网页的标记语言

  • 由标签和被其标记的内容组成，标签的属性控制着显示的方式

    <tag key="value">被标记的内容</tag>

  • 语法不区分大小写，默认使用小写

  • 语法不区分回车、空格和缩进，为了代码可读和逻辑清晰，应该保持严格的缩进风格，必要时添加注释

  • 注释内容<!-- XXXXX --!>，不会被显示

  • 字符字体：保留字符无法直接使用，如< > 会和标签符号产生歧义，可以使用&lt 或 &#60表示小于号

TPlink WDR4310 刷openwrt + 挂USB卡中继WiFi

1、刷机

下载openwrt固件。

找到适合自己设备的固件，使用路由器web升级接口安装。

wdr 4310

有线连接lan口，设置电脑固定ip为192.168.1.x，访问192.168。1.1，登录，系统升级，使用openwrt-install的固件，升级时不要动路由器电源，等待2分钟就好了。

2、安装中文语言包和驱动支持

网络连接

  • 无线：进入op管理页面，网络->无线，找到一个网卡（双频设备会有2个），点击搜索，加入一个可用的无线网络。
  • 有线：网线接入wlan口，在网络->接口，配置拨号连接。

配置SSH连接

在系统->管理权中设置

更新软件源和安装软件

软件包可以在官网download列表中下载，并离线安装。 https://downloads.openwrt.org/chaos_calmer/15.05/ar71xx/generic/packages/

ar71xx表示路由器cpu分支，应选择对应CPU架构的package。

  • 通过web：系统-> 软件包，更新源和输入包名安装软件包
  • 通过ssh： opkg update `opkg install package-name
  • 离线：使用winscp将离线包上传至tmp目录，然后ssh进入系统后：cd /tmp opkg install xxxxxx

安装中文语言支持

  • 离线包：luci-i18n-base-zh-cn
  • opkg install luci-i18n-base-zh-cn
  • 或者通过web安装

安装rt3070驱动

  • 组件：

    kmod-usb-ohci kmod-usb-storage kmod-usb-wdm kmod-usb2

  • 无线驱动：

    kmod-rt2800-pci kmod-rt2800-soc kmod-rt2800-usb

  • lsusb

    opkg install usbutils

挂卡无线中继

首先插入usb无线网卡，lsmod查看一下usb设备，未显示usb网卡设备时，需要检查是否是驱动未装好，usb接口和网卡是否正常。

检测到网卡后，重启一次路由器，ssh reboot，然后重新登入web页面，在网络 -> 无线 下面就可以看到新加入的网卡接口，如果没有，再重启一次试试。
